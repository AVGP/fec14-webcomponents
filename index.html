Recap

  • Web components = reusable, custom HTML elements
  • Standards:
    1. Template elements: Markup for the element
    2. Shadow DOM: Isolation
    3. HTML Imports: Linking & reusing components
    4. Custom Elements: Declaration of your tags
  • Supported in most major browsers (IE and Safari are far behind)
  • Start giving them a try!
